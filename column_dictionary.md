## Improvement Characteristics (`x54s-btds`)

File: `Assessor_-_Single_and_Multi-Family_Improvement_Characteristics_20260624.csv`  
Grain: one row per PIN + CARD + YEAR  
44 columns — include: 23, confirm: 0

- [x] **pin** — Parcel Identification Number (PIN)
- [x] **tax_year** — Tax year
- [x] **card_num** — Card number. Each card is an improvement/building on the parcel
- [ ] **class** — Property class
- [ ] **township_code** — Township code
- [ ] **proration_key_pin** — Tieback key PIN. Prorated properties (whose value is split across multiple PINs) have a "main" or key PIN
- [ ] **pin_proration_rate** — Tieback proration rate. Prorated properties (whose value is split across multiple PINs) pay taxes on the proportion of value on their PIN. In other words, assessed value is multiplied by proration rate to determine taxable assessed value
- [ ] **card_proration_rate** — Card proration rate. Prorated parcels (whose value is split across multiple cards) pay taxes on the proportion of value on their card. In other words, assessed value is multiplied by proration rate to determine taxable assessed value. Cards are divisions within parcels, such as one of multiple buildings on a single parcel.
- [ ] **cdu** — Condition, Desirability, and Utility code. Not well maintained.
- [x] **pin_is_multicard** — Multicard PIN. Indicates whether the parcel contains more than one building (ADU, coach house, etc.)
- [ ] **pin_num_cards** — Number of cards on this parcel. Each card is an improvement/building
- [ ] **pin_is_multiland** — Multiland PIN. Indicates whether parcel has more than one landline
- [ ] **pin_num_landlines** — Number of landlines on a parcel. The sum of all landline square footage should be equal to the total square footage of the parcel. Each landline can correspond to a different land price/rate
- [x] **year_built** — Year built
- [x] **building_sqft** — Building square feet. Square footage of the building, as measured from the exterior
- [x] **land_sqft** — Land square feet. Square footage of the land (not just the building) of the property. Note that a single PIN can have multiple landlines, meaning it can be associated with more than one land price/rate
- [x] **num_bedrooms** — Number of bedrooms
- [x] **num_rooms** — Rooms. Number of total rooms in the building (excluding baths). Not to be confused with bedrooms
- [x] **num_full_baths** — Full baths. Defined as having a bath or shower. If this value is missing, the default value is set to 1
- [x] **num_half_baths** — Half baths. Defined as bathrooms without a shower or bathtub
- [x] **num_fireplaces** — Fireplaces. Counted as the number of flues one can see from the outside of the building
- [x] **type_of_residence** — Type of residence
- [x] **construction_quality** — Construction quality
- [ ] **num_apartments** — Apartments. Number of apartments for class 211 and 212 properties
- [x] **attic_finish** — Attic finish
- [ ] **garage_attached** — Garage 1 attached
- [ ] **garage_area_included** — Is Garage 1 physically included within the building area? If yes, the garage area is subtracted from the building square feet calculation by the field agent
- [x] **garage_size** — Garage 1 size
- [ ] **garage_ext_wall_material** — Garage 1 exterior wall material
- [ ] **attic_type** — Attic type
- [x] **basement_type** — Basement type
- [x] **ext_wall_material** — Exterior wall material
- [x] **central_heating** — Central heating
- [x] **repair_condition** — Repair condition
- [x] **basement_finish** — Basement finish
- [ ] **roof_material** — Roof material
- [x] **single_v_multi_family** — Single vs. multi-family use
- [ ] **site_desirability** — Site desirability
- [ ] **num_commercial_units** — Number of commercial units on the parcel (the vast majority are for properties with class 212)
- [ ] **renovation** — Renovation
- [ ] **porch** — Porch
- [x] **central_air** — Central air conditioning
- [ ] **design_plan** — Design plan
- [ ] **row_id** — Unique row key for API
